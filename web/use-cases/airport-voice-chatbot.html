How the Voice Chatbot Works

1

User Interaction

Passenger speaks or types a query via Airport Chat — the kiosk, mobile app, or web interface

2

Speech Recognition (ASR)

For voice input, Automatic Speech Recognition converts the passenger's spoken query to text

3

Intelligent Query Processing

The LLM interprets intent and determines which Airport Information System to query: Flight Data, Gate/Terminal Status, Ground Transportation, or Airport Services

4

Vector Database (RAG) + API Integrations

RAG retrieves relevant context from the airport knowledge base; API Integrations & Automation fetch live data from connected airport systems

5

LLM Response Generation

The Large Language Model generates a clear, natural-language response from the retrieved data

6

Text-to-Speech Synthesis (TTS)

TTS converts the response to natural-sounding audio, delivered back to the passenger as both voice and text
